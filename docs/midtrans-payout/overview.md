# Overview

Getting started with Payout API

Payouts or Disbursements (previously known as IRIS) is Midtrans' cash management solution that allows you to disburse payment in bulk via API to any bank accounts in Indonesia securely and easily. Payouts API connects to the banks' hosts to enable seamless transfer using integrated APIs.
